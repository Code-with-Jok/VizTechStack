# Architecture Decision Records (ADR)

Store ADR files in this folder using numeric prefixes:

- `0001-short-title.md`
- `0002-short-title.md`

Mandatory sections:

1. Context
2. Decision
3. Alternatives Considered
4. Consequences
5. Rollback Strategy
6. Links (PR/Issue)

When to add/update ADR:

- Any PR with `architecture-impact` label.
- Any change affecting `apps/**`, `convex/**`, `packages/shared/types/**` with cross-module impact.
