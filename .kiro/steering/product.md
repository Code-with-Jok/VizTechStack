# Product Overview

VizTechStack is an interactive learning platform that helps users explore and track technology learning roadmaps. The platform visualizes learning paths as interactive graphs, allowing users to navigate through topics, track progress, and access educational content.

## Core Features

- Interactive roadmap visualization using graph-based UI
- Progress tracking for learning paths
- Content management for guides and topics
- User authentication and role-based access control
- Bookmark and personalization capabilities

## User Roles

- **User**: Browse roadmaps, track progress, bookmark content (read-only)
- **Admin**: Full CRUD access to create and manage roadmaps, topics, and content

## Key Concepts

- **Roadmap**: A learning path with nodes (topics) and edges (dependencies)
- **Category**: role, skill, or best-practice
- **Difficulty**: beginner, intermediate, or advanced
- **Status**: public, draft, or private
